# QueueFlow App Icons

## Required Icons for PWA Installation

To enable "Add to Home Screen" functionality, you need to create the following icon files:

### Icon Sizes Required:
- `icon-72x72.png` (72x72px)
- `icon-96x96.png` (96x96px) 
- `icon-128x128.png` (128x128px)
- `icon-144x144.png` (144x144px)
- `icon-152x152.png` (152x152px)
- `icon-192x192.png` (192x192px) - **Required for Android**
- `icon-384x384.png` (384x384px)
- `icon-512x512.png` (512x512px) - **Required for Android**

### Design Guidelines:
- **Theme**: Queue management, professional service center
- **Colors**: Primary blue (#4f46e5), white background
- **Style**: Clean, modern, recognizable
- **Content**: Could include queue/line icons, numbers, or "Q" letter

### Quick Icon Generation:
1. Create a 512x512px base design
2. Use online tools like:
   - PWA Builder (pwabuilder.com)
   - Favicon Generator (favicon.io)
   - App Icon Generator (appicon.co)

### Temporary Solution:
For testing, you can use placeholder icons or convert any 512x512 image to all required sizes.

### Uganda Context:
Consider incorporating subtle elements that resonate with Uganda's service centers:
- Clean, professional appearance suitable for hospitals, banks, government offices
- Universal queue/service symbols
- Accessible design for all users
